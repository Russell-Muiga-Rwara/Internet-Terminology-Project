FTP (File Transfer Protocol)

Definition

FTP (File Transfer Protocol) is a standard network protocol used to transfer files between a client and a server over the Internet or a local network.

How It Works

A user connects to an FTP server using an FTP client application such as FileZilla. After authenticating with a username and password, the user can upload files from their local computer to the server or download files from the server to their computer. FTP uses two separate connections — one for sending commands and one for transferring the actual file data. SFTP (Secure FTP) is the encrypted, more secure version widely used today.

Why It Matters

FTP is widely used by web developers to upload website files to hosting servers, and by organizations to share large files between departments or with external partners. Before cloud storage services like Google Drive and Dropbox existed, FTP was the primary method of transferring files across networks and it remains in use today in many professional environments.
